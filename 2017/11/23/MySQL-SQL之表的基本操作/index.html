关于外键

如果一个表生成的外键，那么在声明外键之后可以加上ON DELETE|UPDATE {RESTRICT | CASCADE | SET NULL | NO ACTION | SET DEFAULT}以声明在外键所参照的数据被删除或者更新的时候数据库应该采取什么样的策略，其中

  • on delete|upadte是触发事件
  • restrict为显示外键参照对象的修改
  • cascade是当参照对象改变时候级联改变当前表的值
  • set null是设置成nulll
  • no action是不采取任何动作
  • set default是设置为默认值
