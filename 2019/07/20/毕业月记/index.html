毕业月记

发表于 2019-07-20 | | 阅读次数:


（图片为华东师范大学2019届毕业花：绣球花）

距离从学校搬出来住已经差不多一个月了，时间过得飞快，让人回忆无限。想想去年这个时候，自己还在为找工作的事情忙活着，准备简历，刷算法题等等，转眼今天就已经工作快一个月了，甚是感慨！想想自己在研究生三年的生活，很是怀念当时的人和事，现在大家都已经各奔东西，尽管离得不是很远，但能像当时在实验室那样相聚畅谈的日子已经不是很多了。天下没有不散的宴席的理解对于我而言显得更加深刻，这对于我而言同时又更加理解珍惜现在的重要性。希望以后我的同学老师们都能过的很好！
至于工作，我寻求有价值有发展和人性化的工作，至少现在我的工作是符合我的要求的，我会尽力做好我的工作。既然选择了就要做好，这是我的基本原则。希望自己以后能够有更大的发展。
博客从去年找工作断了之后就一致没有拾起来，突然发现有人在提问，但是自己没有能及时回复，所以，自己摸索了半天，加了邮件订阅&RSS订阅和评论邮件通知，邮件订阅&RSS订阅会在有更新的时候再周六周日给订阅者发送邮件，评论邮件通知会在有评论的时候邮件通知我，或者有人回复你的时候邮件通知你(前提是你留了邮箱)，以后会坚持更新博客！加油！
